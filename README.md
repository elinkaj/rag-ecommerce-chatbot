# Online Shop RAG Chatbot

AI-powered shopping assistant built with Streamlit, OpenAI embeddings, PostgreSQL, and pgvector.

## Features

- Semantic product search using embeddings
- RAG (Retrieval-Augmented Generation)
- PostgreSQL + pgvector similarity search
- GPT-4o-mini response generation
- Conversational chat interface with Streamlit
- Live product retrieval from database

## Tech Stack

- Python
- Streamlit
- PostgreSQL
- pgvector
- OpenAI API
- LangChain
- psycopg2
